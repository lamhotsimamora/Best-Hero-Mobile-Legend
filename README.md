## Example Templating Syntax Garuda Javascript 2

### Syntax Templating
Defined the variabel with character # in first and last character.
-------------------
HTML
```
<h1> Hello #username# ! </h1>
```
Javascript
```
__({
 el:"@h1",
 html:{
   single:{
      username:"Lorem Ipsum"
   }
 }
});
```
